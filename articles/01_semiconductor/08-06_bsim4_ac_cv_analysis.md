## はじめに

前回の記事では **DC解析（V–I 特性）** を扱いました。  
しかし MOSFET は、**直流だけで使われるデバイスではありません**。

- 高速動作
- アナログ回路
- 周波数特性

これらを考えると、**容量と周波数応答** を無視することはできません。

本記事では、BSIM4 を用いた **AC解析 / CV解析** を通じて、  
MOSFET の小信号特性を整理します。

---

## AC解析とCV解析の違い

### AC解析
- 小信号を重ねて周波数応答を見る
- ゲイン・位相・gm・ro などを評価

### CV解析
- MOSFET に内在する容量成分を見る
- \(C_{gs}\), \(C_{gd}\), \(C_{gb}\) などを評価

両者は、**高速回路の挙動を理解するための基礎** です。

---

## 使用する環境（SemiDevKit）

本記事では、以下のモジュールを使用します。

- BSIM4 AC / CV Analyzer  
  https://samizo-aitl.github.io/SemiDevKit/bsim/bsim4_analyzer_cv/

前提条件：
- BSIM4 モデルカード（Paramus で生成）
- ngspice
- Python 実行環境

---

## AC解析：周波数応答を見る

### 実行方法

```bash
cd bsim/bsim4_analyzer_cv/run
python run_ac.py
```

この解析では、MOSFET に微小な交流信号を与え、  
周波数を掃引して応答を調べます。

---

### AC解析で分かること

- 小信号ゲイン
- 位相特性
- 周波数帯域
- トランスコンダクタンス \(g_m\)

DC解析で得た動作点の周りで、  
**「どれだけ速く応答できるか」** が見えてきます。

---

## CV解析：寄生容量を見る

### 実行方法

```bash
cd bsim/bsim4_analyzer_cv/run
python run_cv.py
```

CV解析では、端子間の容量成分を評価します。

---

### CV解析で分かること

- ゲート–ソース容量 \(C_{gs}\)
- ゲート–ドレイン容量 \(C_{gd}\)
- バイアス条件による容量変化

これらは、

- 遅延時間
- スイッチング速度
- 周波数特性

に直接影響します。

---

## 寄生容量はどこから来るか

寄生容量は、次の要素から生まれます。

- ゲート酸化膜
- チャネル形成
- オーバーラップ構造
- 空乏層

これらはすべて、**TCAD で見た電位・キャリア分布の結果** です。

---

## DC解析との関係

重要なのは、

> **AC / CV解析は DC 動作点に強く依存する**

という点です。

- DC が不正 → AC / CV も意味を失う  
- DC が妥当 → 小信号解析が生きる  

そのため、**DC解析 → AC/CV解析** の順番が重要になります。

---

## まとめ

- AC解析は周波数応答を見るための解析  
- CV解析は寄生容量を評価する解析  
- 高速・アナログ回路では不可欠  
- DC解析の理解が前提になる
